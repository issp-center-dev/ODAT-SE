================================
Tools
================================

This section introduces the utility tools and post-processing tools included with ODAT-SE.

:doc:`/tool`
    Describes how to use the ``odatse_neighborlist`` command, which generates neighbor list definition files required for Monte Carlo searches in discrete spaces.

:doc:`/post/index`
    A collection of post-processing tools for analyzing PAMC calculation results. Includes log file splitting/aggregation, histogram creation, and model evidence calculation.
