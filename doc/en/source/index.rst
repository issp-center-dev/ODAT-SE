ODAT-SE Documentation
========================

ODAT-SE is a software framework for inverse problem analysis that combines search algorithms with direct problem solvers.
It lets you try multiple search methods within a common workflow for inputs, outputs, and execution.

--------------------------

- :doc:`Run a Sample First <tutorial/index>`
  Follow hands-on examples to understand how each algorithm works in practice.
- :doc:`Look Up Input and Output Details <manual/index>`
  See the specifications of input files, output files, algorithms, and solvers.
- :doc:`Troubleshoot a Problem <faq/index>`
  Find answers on installation, algorithms, errors, and customization.

Overview
--------

- Combine search algorithms and direct problem solvers to solve inverse problems
- Choose among Bayes, PAMC, replica exchange, grid search, and other methods
- Extend the framework with custom solvers or external programs
- Use sample data and templates as a starting point for real applications
