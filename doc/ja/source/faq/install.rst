========================================
インストール・実行
========================================

Python 3.8 以前で動きますか？
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Python 3.9 以上が必要です。``python3 --version`` でバージョンを確認してください。


``odatse`` コマンドが見つかりません
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

``pip install`` 時にインストールされた場所が PATH に含まれていない可能性があります。
以下を確認してください。

.. code-block:: bash

    # インストール状況の確認
    $ python3 -m pip show ODAT-SE

    # pip のインストール先を確認
    $ python3 -m pip show ODAT-SE | grep Location

``--user`` オプションでインストールした場合は ``~/.local/bin`` が PATH に含まれているか確認してください。
どうしてもコマンドが見つからない場合は、直接実行できます。

.. code-block:: bash

    $ python3 -m odatse input.toml


MPI を使わないで実行できますか？
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

はい、多くのアルゴリズムは MPI なしでシリアル実行できます。
mpi4py がインストールされていなくても自動的にシリアルモードになります。

ただし、以下のアルゴリズムは MPI 並列を前提としています。

- ``exchange``: レプリカ交換にはレプリカ数以上のプロセスが必要です
- ``mapper``: MPI 並列でグリッド点を分担します（1プロセスでも動作しますが、大規模なグリッドでは非常に遅くなります）
